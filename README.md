Webrnote
========
About
--------------
Simple GTK+ 3 Webview App for Evernote  
v 0.2 - (Please check requirements, moved to GTK+ 3)

Requirements
---------------------
- Python
- gir1.2-webkit-3.0
- Also an Evernote account

Runing
--------------
	git clone https://github.com/dbtek/webrnote.git
	cd webrnote
	./webrnote.py
